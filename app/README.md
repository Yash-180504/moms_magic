# Mom's Magic App (Expo)

Mobile app workspace for Mom's Magic.

## Stack
- React Native
- Expo
- NativeWind
- lucide-react-native icons

## Setup
1. `cd /Users/devil_2012/Desktop/moms_magic/app`
2. `npm install`
3. `npm start`

Then run on:
- Android emulator/device (`a` in Expo CLI)
- iOS simulator (`i` in Expo CLI on macOS)
- Expo Go by scanning QR code

## Structure
- `App.js`: home marketplace screen
- `src/components`: reusable UI blocks
- `src/data/mockProviders.js`: demo provider/category data
- `tailwind.config.js`: NativeWind theme tokens
